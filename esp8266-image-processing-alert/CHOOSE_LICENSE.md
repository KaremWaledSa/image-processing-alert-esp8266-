# Choose a license before publishing

I did **not** force a license choice for you.

Reason:
- a public GitHub repository does **not** automatically mean you want MIT/Apache/GPL
- the license is a legal choice and should match how you want others to use your code

Recommended:
- MIT: easiest for portfolio/public demos
- Apache-2.0: permissive with patent language
- GPL-3.0: if you want derivatives to stay open

Replace this file with a real `LICENSE` file before public release if you want GitHub to detect it.
